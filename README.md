# ✍️ DocConvert: Premium Rich Text & Markdown Editor

**DocConvert** is a modern, high-performance web tool designed for seamless conversion between Rich Text and Markdown. Built with a focus on premium aesthetics and user privacy, it features integrated AI capabilities powered by Google Gemini.

## ✨ Key Features

- 🔄 **Bidirectional Sync**: Convert Rich Text to Markdown (and vice versa) in real-time.
- 🪄 **AI Polish**: Automatically refine your writing for clarity, grammar, and style with built-in AI.
- 📝 **AI Summarize**: Summarize long documents into a single paragraph instantly.
- 📄 **PDF Import**: Upload PDFs and convert them directly to clean Markdown using AI.
- 📑 **Live Preview**: See your changes happen instantly across editors.
- 📥 **One-Click Export**: Save your work directly as a `.md` file.
- 🔒 **Privacy-First**: Your Gemini API key is stored locally in your browser and never sent to our servers.

---

## 🚀 Getting Started

### Prerequisites
To use the AI features (Polish, Summarize, PDF Import), you'll need a free Gemini API key:
1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Create and copy your API key.

### How to Use
1. Open `index.html` in your browser (or visit the live link).
2. Click the ⚙️ **Settings** icon.
3. Paste your Gemini API key and click **Save Configuration**.
4. You're ready to start writing!

---

## 🛡️ Privacy
We take privacy seriously. All documents and API keys are stored within your browser's `LocalStorage`. No data is ever collected or sent to a third-party server, other than direct requests to Google's Gemini API to power the smart features.
